About This Project

The Resume Categorization System is an AI-driven final-year engineering project designed to simplify the recruitment process. Using advanced NLP techniques, the system scans resumes, extracts important keywords, analyzes role-related information, and classifies them into categories like:

  • Data Scientist
  • Machine Learning Engineer
  • Software / Web Developer
  • Cloud / DevOps Engineer
  • Data Analyst
  • Cybersecurity Analyst

This system helps companies reduce hiring time, improves accuracy, and eliminates manual resume shortlisting.

Key Features

📄 Resume Parsing

Extracts skills, experience, education, projects, and important keywords from resumes.

🧠 AI/ML Classification

Uses machine learning to assign each resume to the most appropriate job category.

⚡ Fast Processing

Handles multiple resume uploads at once with fast and accurate results.

☁ Flask Web Interface

Simple and user-friendly web UI built using Flask for interactive analysis.

📊 Insights

Displays categorized job roles and important extracted information.

🔍 NLP Techniques

Implements TF-IDF, bag of words, tokenization, and vectorization techniques.

      How the System Works

      1️⃣ Upload Resume

      Upload one or multiple files at once using drag & drop or browse.

      2️⃣ Text Extraction

      The system reads the resume text using PDF readers / docx parsers.

      3️⃣ NLP Processing

      Applies tokenization, TF-IDF vectorization, and keyword extraction.

      4️⃣ ML Classification

      Predicts the most relevant job role using trained ML models.

      5️⃣ Display Results

      Shows categorized output instantly with clear and readable UI.
